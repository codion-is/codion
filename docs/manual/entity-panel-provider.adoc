== Introduction

The {url-javadoc}/org/jminor/swing/framework/ui/EntityPanelProvider.html[EntityPanelProvider] class provides lazy initialization for {url-javadoc}/org/jminor/swing/framework/ui/EntityPanel.html[EntityPanels]. Using the *EntityPanelProvider* class instead of instantiating *EntityPanels* directly means the panels are not initialized until made visible.
